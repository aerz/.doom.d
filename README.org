#+title: .doom.d

My personal [[https://github.com/doomemacs/doomemacs][Doom Emacs]] configuration. Built for a fast, frictionless editing experience.

** Installation

*** Emacs

Installed via [[https://github.com/aerz/nix][Nix]], using [[https://github.com/d12frosted/homebrew-emacs-plus][emacs-plus]] for native macOS support.

*** Doom Emacs

#+begin_src sh
git clone https://github.com/aerz/.doom.d ~/.config/doom
git clone --depth 1 --single-branch https://github.com/doomemacs/doomemacs ~/.config/emacs
~/.config/emacs/bin/doom install
#+end_src

** Commands

Essential Doom commands to keep nearby.

- ~doom sync~ — apply changes after editing config
- ~doom upgrade~ — update Doom and packages
- ~doom doctor~ — diagnose issues
