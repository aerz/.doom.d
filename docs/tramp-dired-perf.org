#+title: TRAMP Dired Performance
#+author: aerz
#+date: 2026-06-22

** Overview

Cursor movement in Dired over SSH triggers VC/project hooks that make remote
round-trips, causing per-line lag.

** Configuration

The [[https://github.com/doomemacs/modules/blob/8966104447d0a3131a232187bde148f58b32a322/modules/emacs/tramp/config.el][:emacs tramp]] module handles most of it (caching, async processes,
ControlMaster hooks). The remaining fix is ensuring SSH reuses connections via
~~/.ssh/config~:

#+begin_src conf
Host *
  ControlMaster auto
  ControlPath ~/.ssh/sockets/%r@%h:%p
  ControlPersist 10m
#+end_src

#+begin_src sh
mkdir -p ~/.ssh/sockets
#+end_src
